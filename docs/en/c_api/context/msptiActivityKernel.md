# msptiActivityKernel<a name="ZH-CN_TOPIC_0000002045983217"></a>

msptiActivityKernel is the structure corresponding to the activity record type [MSPTI\_ACTIVITY\_KIND\_KERNEL](msptiActivityKind.md), defined as follows:

```cpp
typedef struct PACKED_ALIGNMENT {
    msptiActivityKind kind;   // Activity record type MSPTI_ACTIVITY_KIND_KERNEL
    uint64_t start;   // Start timestamp of Kernel execution on the NPU device, in ns. If both the start and end timestamps are 0, the Kernel timestamp information cannot be collected
    uint64_t end;   // End timestamp of Kernel execution, in ns. If both the start and end timestamps are 0, the Kernel timestamp information cannot be collected
    struct {
        uint32_t deviceId;   // Device ID of the device on which the Kernel runs
        uint32_t streamId;   // Stream ID of the kernel execution stream
    } ds;
    uint64_t correlationId;   // Unique ID generated by the runtime when launching a kernel. Other activities can correlate with the kernel through this value
    const char *type;   // Kernel type
    const char *name;   // Kernel name, which remains consistent throughout the activity record and is not recommended to be modified
} msptiActivityKernel;
```
